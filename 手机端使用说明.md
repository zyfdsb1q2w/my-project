# 美霖个人助手 - 手机端使用指南

## 📱 手机安装方法

### 方法一：PWA安装（推荐）
**支持平台**：Android Chrome、iOS Safari

**安装步骤**：
1. **在手机上打开浏览器**（Chrome或Safari）
2. **访问应用地址**：
   - 确保手机和电脑在同一WiFi网络
   - 在电脑上运行 `python start_ai.py`
   - 查看终端输出的"Network URL"（如：http://192.168.x.x:8501）
   - 在手机浏览器中输入该URL
3. **安装PWA应用**：
   - **Android Chrome**：点击地址栏右侧的"安装"图标 ⬇️
   - **iOS Safari**：点击分享按钮 📤 → 添加到主屏幕
4. **完成安装**：应用图标将出现在手机主屏幕

**优点**：
- 类似原生应用的体验
- 可以离线使用部分功能
- 无需下载安装包
- 自动更新

### 方法二：通过电脑中转访问
**适用场景**：临时使用，不想安装

**步骤**：
1. 在电脑上运行 `python start_ai.py`
2. 查看终端输出的"Network URL"
3. 在手机浏览器中输入该URL
4. 直接使用，无需安装

### 方法三：使用打包的exe文件（Windows电脑）
**适用场景**：在Windows电脑上使用

**步骤**：
1. 下载 `美霖个人助手_带图标_分发包.zip`
2. 解压到任意文件夹
3. 编辑 `.env` 文件，添加DeepSeek API密钥
4. 双击 `美霖个人助手.exe` 运行
5. 在手机浏览器中访问电脑的IP地址和端口

## ❓ 常见问题

### Q1: 手机无法访问应用
- 检查手机和电脑是否在同一WiFi网络
- 检查电脑防火墙设置
- 尝试关闭杀毒软件临时测试

### Q2: PWA安装按钮不显示
- 确保使用Chrome（Android）或Safari（iOS）
- 清除浏览器缓存
- 等待页面完全加载

### Q3: AI无法回答问题
- 检查 `.env` 文件中的API密钥是否正确
- 确认网络连接正常
- 确保API密钥有足够的余额

### Q4: 应用运行缓慢
- 检查网络连接速度
- 关闭其他后台应用
- 清除浏览器缓存

### Q5: 如何备份数据
- 应用数据存储在浏览器本地
- 可以导出重要记录到文本文件
- 定期截图保存重要信息
