# 美霖个人助手 - PWA手机应用

一个基于DeepSeek API的个人AI助手，专为手机端优化的渐进式Web应用（PWA）。

## ✨ 功能特点

### 🤖 AI智能助手
- 集成DeepSeek API，提供智能对话
- 支持自定义AI行为规则
- 实时聊天，自然语言交互

### 💖 生活记录
- **每日土味情话**：每天自动更新，可收藏喜欢的情话
- **今日想法**：记录每日思考和计划
- **今日目标**：设定并追踪每日目标
- **心情记录**：追踪每日心情变化
- **记事本**：记录重要备忘和笔记

### 📱 移动端优化
- **PWA支持**：可安装到手机主屏幕，类似原生应用
- **响应式设计**：自动适配手机屏幕
- **触摸优化**：大按钮、大字体，便于手机操作
- **离线功能**：部分功能支持离线使用

### ⚙️ 个性化设置
- 自定义AI助手规则
- 个性化生活建议
- 数据本地存储，保护隐私

## 🚀 快速开始

### 环境要求
- Python 3.8+
- 稳定的网络连接
- DeepSeek API密钥

### 安装步骤

1. **克隆或下载项目**
   ```bash
   git clone <项目地址>
   cd MyPersonalAI
   ```

2. **安装依赖**
   ```bash
   pip install -r requirements.txt
   ```

3. **配置API密钥**
   - 访问 [DeepSeek平台](https://platform.deepseek.com/) 获取API密钥
   - 创建 `.env` 文件，添加以下内容：
     ```
     DEEPSEEK_API_KEY=your_api_key_here
     ```

4. **启动应用**
   ```bash
   python start_pwa.py
   ```

## 📲 手机安装指南

### PWA安装步骤

1. **在电脑上启动应用**
   ```bash
   python start_pwa.py
   ```

2. **获取手机访问地址**
   - 查看终端输出的"Network URL"
   - 格式如：`http://192.168.x.x:8501`

3. **手机访问并安装**
   - **Android Chrome**：
     1. 在手机Chrome浏览器中输入URL
     2. 点击地址栏右侧的"安装"图标 ⬇️
     3. 确认安装
   
   - **iOS Safari**：
     1. 在手机Safari浏览器中输入URL
     2. 点击分享按钮 📤
     3. 选择"添加到主屏幕"
     4. 确认添加

4. **完成安装**
   - 应用图标将出现在手机主屏幕
   - 点击图标即可打开应用，类似原生应用体验

### 网络要求
- 手机和电脑必须在同一WiFi网络
- 需要网络连接调用AI API
- 部分界面资源可离线使用

## ❓ 常见问题

### Q1: 手机无法访问应用
- 检查手机和电脑是否在同一WiFi
- 检查电脑防火墙设置
- 确认Streamlit服务已启动

### Q2: PWA安装按钮不显示
- 使用Chrome（Android）或Safari（iOS）
- 清除浏览器缓存
- 等待页面完全加载

### Q3: AI无法回答问题
- 检查 `.env` 文件中的API密钥
- 确认网络连接正常
- 确保API密钥有足够余额

### Q4: 如何备份数据
- 应用数据存储在浏览器本地
- 可导出重要记录
- 定期截图保存
